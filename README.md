# Run Scripts
To download neseccary dependencies, in the project directory run:
`npm install`

Then do one of the folowing, 
`yarn start`
`npm start`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
